## 📌 Overview
This project is based on **Python Basics & Control Flow**.

It includes 4 tasks:
1. Data Cleaning & Parsing
2. Marks Analysis using loops

All tasks are implemented using **Jupyter Notebook (.ipynb)**.

# 🧩 Task 1 — Data Parsing & Profile Cleaning

## 🎯 Objective
- Clean raw student data
- Convert data types
- Validate names
- Display formatted student profiles

## ⚙️ What I Did
- Removed extra spaces using `.strip()`
- Converted names to Title Case using `.title()`
- Converted roll numbers to integers
- Converted marks string to list
- Checked name validity using `.isalpha()`
- Printed formatted student profile cards
